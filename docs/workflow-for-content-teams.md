# Workflow For Content Teams

## Publishing Preflight

1. Gather candidate videos in one local folder.
2. Run the scanner.
3. Review low-risk files first.
4. Assign medium-risk groups to a human reviewer.
5. Send high-risk or repeated groups to editors for stronger changes.
6. Keep `report.md` with the publishing batch.

## Editing Team Review

Use the report to identify exports that may be too close to each other. The report should guide discussion, not replace editor judgment.

## Batch Cleanup

For historical folders, use exact duplicate groups to find storage waste and near-duplicate groups to decide which clips should be archived, re-edited, or kept. Do not delete files automatically.

## Review Questions

- Are multiple files the same product angle with only compression changes?
- Are subtitles, intros, or end cards the only difference?
- Does the video contain materially different footage?
- Does the team need a new script, shooting angle, scene order, or voiceover?
